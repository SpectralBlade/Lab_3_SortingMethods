## Пользовательские допущения (взаимодействие с программой)
Пользователь вводит команду/функцию, которую он хочет выполнить. 
1. Если это команда сортировки: пользователь вводит нужную ему сортировку (например, bucket_sort), затем поочередно для нее данные (желаемый массив, после ключ/компаратор)
2. Если это математическая функция: пользователь может ввести как функцию с аргументами сразу (например, factorial 5), либо же сначала функцию, затем ее аргумент.
3. Если это команда для структуры данных, а именно - добавление в стек (stack_push) или в очередь (queue_enqueue): пользователь вводит и команду, и добавляемое значение сразу.

Допущения:
1. Одновременно может существовать только один стек/одна очередь (при попытке вызвать команду создания - старая структура перезапишется новой пустой)

# Реализованные команды (такое же сообщение показывается пользователю при введении help)

## Команды для сортировки:
1. bubble_sort - сортировка пузырьком (самая медленная, сравнение 2 чисел)
2. quick_sort - быстрая сортировка (относительно pivot)
3. heap_sort - пирамидальная сортировка (кучей)
4. counting_sort - сортировка подсчетом (*)
5. bucket_sort - сортировка ведрами (* **)
6. radix_sort - подразрядная сортировка (*)

(\*) - не поддерживают строки (только натуральные числа + 0), (**) - поддерживает только числа в диапазоне [0, 1)

Все элементы в сортировках должны быть одного типа (int/str/float)

## Ключи в сортировках:
1. abs - сравнение без знака 
2. neg - сравнение в обратном порядке 
3. str - сравнение в строковом представлении 
4. len - сравнение относительно длины

## Компараторы в сортировках:
1. desc - сравнение по убыванию (в обратном порядке)
2. abs - сравнение по модулю (без знака)
3. str - сравнение в строковом представлении

## Команды для стека:
1. create_stack - создать новый пустой стек (заменить текущий)
2. stack_push X - добавить элемент X в стек 
3. stack_pop - удалить верхний элемент стека и вывести его 
4. stack_peek - вывести элемент на вершине стека без удаления 
5. stack_is_empty - проверить, пуст ли стек (возвращает True/False)
6. stack_len - вывести количество элементов в стеке 
7. stack_min - вывести минимальный элемент в стеке 
8. create_stack - создать новый пустой стек

Команды 2-8 не работают без созданного ранее стека (выведут ошибку)

## Команды для очереди:
1. create_queue - создать новую пустую очередь (заменить текущую)
2. queue_enqueue X - добавить элемент X в очередь (в конец)
3. queue_dequeue - удалить первый элемент очереди и вывести его 
4. queue_front - вывести первый элемент без удаления 
5. queue_is_empty - проверить, пуста ли очередь (возвращает True/False)
6. queue_len - вывести количество элементов в очереди 
7. create_queue - создать новую пустую очередь

Команды 2-7 не работают без созданной ранее очереди (выведут ошибку)

## Математические функции:
1. factorial X - вычислить факториал от X
2. fact_recursive X - вычислить факториал от X рекурсивным методом
3. fibonacci X - вычислить X-ое число Фибоначчи 
4. fibo_recursive X - вычислить X-ое число Фибоначчи рекурсивным методом
